An '''achievement gallery''' is a room in a [[player-owned house]] in which the player can display their various accomplishments. It can be created at level 80 [[Construction]] for {{Coins|200000}}. A player may only have one achievement gallery in their POH at any given time.

If the player uses an orange [[spicy stew]] to boost their Construction by 5 levels, it is possible to build an achievement gallery at level 75 Construction. Likewise a [[Cup of tea (trimmed)]] boost at 77 Construction. The [[crystal saw]]'s +3 boost cannot be used to build the achievement gallery, since saws are not used when building new rooms.

There are 6 [[hotspot]]s available:
*[[Adventure log space]]
*[[Altar space#Achievement Gallery|Altar space]]
*[[Boss lair space]]
*[[Display space]]
*[[Jewellery box space]]
*[[Quest list space]]

==Boss lair==
The boss lair allows a player to build a fancy display showing off various [[boss]]es. In order to display a boss lair, players must have killed the boss once, and use the jar dropped by that boss on the display.

The display can be configured to show any boss whose jar has been added to it. Once put in the display, players can remove the jar should they wish to do so, but must put the jar back if they wish to display that boss again.

==Jewellery box==
The jewellery box can store various teleportation [[jewellery]]. Each box will provide unlimited teleports to locations provided by the enchanted jewellery.

Jewellery boxes are constructed in upgrades: before a higher level box can be built, the player must first build the one before it. Bank notes are NOT accepted, each item must be brought.
